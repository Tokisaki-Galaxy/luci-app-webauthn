如何使用 (How to use)

在你的 OpenWrt 终端运行以下命令：

# 1. 添加公钥
wget https://tokisaki-galaxy.github.io/luci-app-webauthn/all/key-build.pub -O /tmp/key-build.pub
opkg-key add /tmp/key-build.pub

# 2. 添加软件源
echo "src/gz luci-app-webauthn https://tokisaki-galaxy.github.io/luci-app-webauthn/all" >> /etc/opkg/customfeeds.conf

# 3. 更新并安装
opkg update
opkg install luci-app-webauthn
